# CLAUDE.md — AdamPartsFinder

## Overview
Proof of concept price comparison tool for supplier parts. Built as an R Shiny application deployed via Docker Compose.

## Tech Stack
- R (Shiny framework)
- Docker / Docker Compose

## Development
```bash
# Run locally with R
Rscript -e "shiny::runApp('.')"

# Run with Docker
docker compose up -d
```

The app is structured as a standard Shiny app with `global.R`, `server.R`, and `ui.R` entry points.

## Architecture
- `global.R` — shared data and configuration loaded at startup
- `server.R` — server-side logic and reactivity
- `ui.R` — user interface definition
- `www/` — static assets (CSS, JS, images)
- `data/` — supplier data files
- `docs/` — documentation and images
